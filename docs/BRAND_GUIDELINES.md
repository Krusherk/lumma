# Lumma Brand Guidelines

## Brand Core
- Name: `Lumma`
- Positioning: Stablecoin utility with game loops.
- Voice: Credible + playful.
- Mandatory lockup: include `Built on Arc` in app footer and X bio.

## Visual System
- Primary ink: `#0E1116`
- Accent sky: `#5EE9FF`
- Accent lime: `#C6FF5C`
- Sand base: `#F7F4EA`
- Alert: `#FF6B6B`

## Typography
- Display: `Space Grotesk`
- Body/UI: `Manrope`

## Logo Usage
- Primary mark: liquid eclipse + star fracture.
- Keep clear space equal to height of the inner eclipse.
- Minimum digital size: `24px`.
- Do not recolor with rainbow gradients.

## UX Messaging Rules
- Any APY claim on testnet must include `Estimated APY (testnet model)`.
- Avoid return guarantees and certainty language.
- Keep risk labels visible for each vault tier.
